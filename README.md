# Student Performance Analysis

End-to-end data analysis and machine learning project using Python and Jupyter Notebook to analyze and predict student performance based on academic and demographic features.

## Dataset
- Source: Kaggle – Students Performance in Exams
- File: StudentsPerformance.csv

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Project Workflow
1. Data Ingestion
2. Data Cleaning & Feature Engineering
3. Exploratory Data Analysis (EDA)
4. Machine Learning Models
5. Model Evaluation

## Machine Learning Models Used
- Linear Regression
- Random Forest Regressor

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

## Conclusion
This project demonstrates a complete data analysis and machine learning pipeline and highlights key factors influencing student academic performance.
